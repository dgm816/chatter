# Chatter

A simple IRC client written in C.

## Build Instructions

This project uses CMake for its build system. To build the project, follow these steps:

1. Create a build directory:

   ```bash
   mkdir build
   cd build
   ```

2. Run CMake to configure the project:

   ```bash
   cmake ..
   ```

3. Build the project:

   ```bash
   make
   ```
